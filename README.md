# mcu-iot-device-v1
iot


项目简介：
MCU采集温湿度传感器的数据，通过液晶屏显示，并通过涂鸦WiFi模块上传到手机APP端显示。通过红外发送器可以远程控制空调开关，进一步开发可以根据温湿度传感器的数据控制空调的运行状态，达到更好的温控效果。

使用教程：
设备上电，按下网络配置按键，使黄灯闪烁，打开手机APP链接设备并配置网络。通过APP中的设置中的开关可以远程控制空调的开关。
